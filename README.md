# Ecommerce-website              
This project is a simple E-Commerce Website developed using HTML, CSS, and JavaScript. The main goal of this project is to create a responsive and user-friendly online shopping interface. HTML is used to build the structure of the website, CSS is used to design and style the pages, and JavaScript is used to add interactivity such as product display, add-to-cart functionality, and dynamic updates. The website includes features like product listings, a shopping cart system, and basic user interaction to simulate a real online shopping experience. This project helped me understand front-end development concepts and improve my skills in building interactive web applications.

# E-Commerce Website 🛒
